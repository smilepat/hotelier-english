# 호텔리어 고객 응대 영어 훈련 앱 PRD (Refined)

- 작성일: 2026-04-09
- 버전: v1.1
- 키워드: #호텔영어 #응대훈련 #시나리오
- 기술 스택: Vanilla HTML/CSS/JS (Single Page Application)

## 1) 제품 목표
호텔 프론트/컨시어지/객실 서비스 담당자가 실제 고객 질문에 대해 **자연스럽고 정확한 영어 응대**를 빠르게 연습하도록 지원한다.

## 2) 타깃 사용자
- 신규 호텔리어(입사 1년 미만)
- 비영어권 호텔 스태프
- 서비스 품질 코칭 담당 매니저

## 3) 핵심 문제
- 상황별 응대 문장을 외워도 실제 대화에서 즉시 꺼내기 어렵다.
- 발화 정확도(표현/정중함/완결성) 피드백을 즉시 받기 어렵다.
- 개인별 취약 주제를 추적하기 어렵다.

## 4) 핵심 가치 제안
- **시나리오 기반 반복 훈련**으로 실전 대처 능력 강화
- **즉각 피드백**으로 표현 정확도와 자신감 동시 향상
- **진행 추적**으로 주제별 취약점 확인 및 개선

## 5) 기능 요구사항 (MVP)

### 5.1 시나리오 학습
- 총 20개 기본 시나리오 제공
- 주제: 체크인, 객실 요청, 시설 안내, 교통/관광, 불만 처리, 정산, 체크아웃, 긴급, 일반
- 각 시나리오 구성:
  - 고객 질문(영문)
  - 추천 응답(모범 답안)
  - 변형 표현(대체 문장 3개)
  - 부적절 응답(bad option 1개)

### 5.2 응답 방식
- **객관식 모드**: 4지선다 중 적절한 응답 선택
- **자유응답 모드**: 텍스트 입력 후 핵심 표현 포함 여부 평가
- **TTS 지원**: 고객 질문을 음성으로 재생 (Web Speech API)

### 5.3 즉각 피드백
- 정답/오답 여부 즉시 표시
- 왜 좋은 표현인지(정중함, 정보 완결성, 행동 제시) 안내
- 모범 답안 + 대체 표현 노출

### 5.4 진행 추적 (Dashboard)
- 전체 완료율 (프로그레스 바)
- 정확도 퍼센트
- 주제별 완료 상태
- 취약 문항(오답) 목록 + 시도 횟수

### 5.5 UX 요구사항
- 다크 럭셔리 테마 (호텔 분위기)
- 시나리오 간 네비게이션 (이전/다음)
- 주제별 필터링
- 3클릭 이내 학습 시작
- 반응형 레이아웃 (모바일/데스크톱)

## 6) 비기능 요구사항
- 외부 의존성 없음 (CDN 불필요)
- localStorage로 학습 상태 영구 저장
- 부드러운 애니메이션 / 마이크로 인터랙션
- 접근성 고려 (키보드, 스크린리더)

## 7) 성공 지표 (KPI)
- 시나리오 정답률 평균 20%p 향상(4주)
- 재방문율(주 2회 이상) 50%+
- 사후 설문: "영어 응대 자신감 상승" 응답 4.2/5 이상

## 8) 향후 확장
- Web Speech API 음성 인식 → 발음 평가
- 난이도 자동 조절(초급/중급/고급)
- 실제 호텔 SOP 맞춤 시나리오 업로드
- 다국어 UI (한국어/일본어/중국어)
- Leaderboard / 팀 대시보드

## 9) 초기 콘텐츠(20개) 운영 원칙
- 모든 답변은 정중 표현 포함(예: Certainly, I'd be happy to...)
- 행동/다음 단계 제시(예: I'll arrange..., Let me check...)
- 시간/위치/대안 정보 제공 우선
